3D SHAPES


14 November 2019 | Javascript , p5.js

For making the 3D shapes ,we need to add the WEBGL parameter in the createCanvas().

Box


box(with,height,depth)



Ellipsoid


ellipsoid(radiusX,radiusY,radiusZ, detailX, detailY)



Torus


torus(radius,tuberadius,detailX,detailY)



Cone


cone(radius,height,detailX,detailY)



Sphere


sphere(radius)



To fill the colour in the 3D shapes ,the simplest way is to write normalMaterial() function. The colours will automatically appear on the 3D object.



But these shapes doesn't look like 3D ,to make them look like 3D we need to rotate our shapes.For that ,first make the global variable of angle and initialise it with zero. Then in the draw function , update the angle . In WEBGL rotate function works as rotateX, rotateY and rotateZ. Try rotating in different axis separately to see how they move.

rotateX


rotateY


rotateZ


Now, in this rotate function put the variable of angle ,to make the shapes rotate.
